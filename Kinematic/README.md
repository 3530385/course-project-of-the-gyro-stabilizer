На кинематической схеме показана структура, системы координат, состав гиростабилизатора и электрические связи между его элементами. Гиростабилизатор выполнен по двухосной схеме. При этом углы прокачки по обеим осям лежат в пределах от -50° до +50°. По оси наружной рамки применен консольный подвес. По роли гироскопа в процессе стабилизации, проектируемый гиростабилизатор – индикаторный.Системы координат, заданные для гиростабилизатора: 
![image](https://user-images.githubusercontent.com/45338087/178803168-5197c5ae-b52f-48e7-9176-c99557f99e07.png)

Системы координат, связанные с ГС:
- $X_c,Y_c,Z_c$- оси, связанные с основанием (с ЛА);
- $X_p,Y_p,Z_p$- оси, связанные с наружной рамкой;
- $X_п,Y_п,Z_п$- оси, связанные с платформой;
- $α$ и $β$ – углы поворота: наружной рамки ГС относительно основания и платформы вокруг внутренней оси наружной рамки соответственно.
	
Чувствительный элемент – трёхстепенной гироскоп, который установлен на платформе следующим образом:
- Ось наружной рамки гироскопа $Y_г$ сонаправлена с осью $Y_п$;
- Ось внутренней рамки гироскопа $X_г$ сонаправлена с осью $X_п$;
- Ось собственного вращения гироскопа $Z_г$ сонаправлена с осью $Z_п$;

кинематическая схема выполнена в САПР "Компас 3D".
