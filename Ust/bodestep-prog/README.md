# Программа для исследования и подбора постоянных времени регулятора.

Позволяет подбирать постоянные времени регулятора в цепи обратной связи канала стабилизации гиростабилизатора.
Изменяя положение ползунков можно получить оптимальные постоянные времени регулятора, позволяющие получить необходмые параметры переходного процесса.
