# course-project-of-the-gyro-stabilizer
### Этот репозиторий содержит курсовой проект по дисциплине "Теория гироскопов и гиростабилизаторов"
(МГТУ им. Н. Э. Баумана 4 курс).

В данном курсовом проекте разработан двухосный гиростабилизатор оптического прицела вертолёта согласно [техническому заданию](https://github.com/3530385/course-project-of-the-gyro-stabilizer/tree/main/TZ) 

![image](https://user-images.githubusercontent.com/45338087/178764330-c5a01de6-1840-4200-aa42-f1d15058f83d.png)

- [Кинематическую схему гиростабилизатора](https://github.com/3530385/course-project-of-the-gyro-stabilizer/tree/main/Kinematic)
- [Общий вид гиростабилизатора](https://github.com/3530385/course-project-of-the-gyro-stabilizer/tree/main/OB)

- [Расчёт устойчивости каналов стабилизации](https://github.com/3530385/course-project-of-the-gyro-stabilizer/tree/main/Ust)
