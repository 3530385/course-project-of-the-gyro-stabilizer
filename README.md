# course-project-of-the-gyro-stabilizer
### Этот репозиторий содержит курсовой проект по дисциплине "Теория гироскопов и гиростабилизаторов"
(МГТУ им. Н. Э. Баумана 4 курс).

В данном курсовом проекте разработан двухосный гиростабилизатор оптического прицела вертолёта согласно ![техническому заданию] 

![image](https://user-images.githubusercontent.com/45338087/178764330-c5a01de6-1840-4200-aa42-f1d15058f83d.png)

Курсовой проект содержит:
- Расчётно пояснительную записку
- Кинематическую схему гиростабилизатора
- Общий вид гиростабилизатора
- Чертёж чувствительного элемента гиростабилизатора (МГТУ-05)
- Электрическая схема принципиальная
- Расчёт устойчивости каналов стабилизации
- Схему сборки и размерную цепь
